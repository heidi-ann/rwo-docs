Writing values to the binary protocol using unsafe C.
type 'a writer = Unsafe_common.sptr -> Unsafe_common.eptr -> 'a -> Unsafe_common.sptr
Type of unsafe writer functions for the binary protocol. They take a start pointer for writing, an end pointer designating the end of the buffer and the value to be written, and return the start pointer to the next write position.
