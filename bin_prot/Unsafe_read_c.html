type 'a reader = Unsafe_common.sptr_ptr -> Unsafe_common.eptr -> 'a
Type of unsafe reader functions for the binary protocol. They take a pointer to a source pointer to start reading and an end pointer designating the end of the buffer, and return the unmarshalled value. The pointer to the next buffer position after reading in the value will be stored in the pointer to the source pointer.
