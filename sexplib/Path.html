Path: Module for Substitutions within S-expressions


Types


type el =
|Pos of intPos n denotes nth element in a tuple
|Match of string * intMatch (tag, n) denotes nth argument of sum matching tag
|Rec of stringRec name denotes the record field having name
Type of substitution elements
type t = el list
