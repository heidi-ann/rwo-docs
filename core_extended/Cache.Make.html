type ('k, 'v) t
A key value cache
type 'a with_init_args
Used to specify the type of the Cache.S.create and Cache.S.memoize function. This describes the arguments required to initialise the caching strategy and the store. For instance if the store doesn't take any argument (eg.: Cache.Store.Table) and the strategy takes an int (eg.: Cache.Strategy.Lru) this type will be: 

   type 'a with_init_args = int -> 'a
